GUIA RÁPIDO
Checklist de instalação em ordem

ANTES DE COMEÇAR

1

Criar conta no n8n

Acesse app.n8n.io → Sign Up → confirme o email

2

Chave API do Claude

console.anthropic.com → API Keys → Create Key → salve a chave sk-ant-... → adicione U$5 em Billing

3

Google Cloud — APIs e OAuth

Criar projeto → ativar YouTube Data API v3 + Gmail API + Drive API → criar chave de API → criar OAuth Client ID com redirect https://app.n8n.io/rest/oauth2-credential/callback

4

Shotstack

shotstack.io → criar conta → copiar chave sandbox (gratuita para testes)

5

Cloudinary

cloudinary.com → criar conta → copiar Cloud Name + API Key + API Secret → criar upload preset chamado lacalle sem assinatura

GOOGLE DRIVE

6

Estrutura de pastas

Criar pasta NOME_DO_CANAL → dentro criar subpastas com data DDMMAAAA (ex: 17032026) → colocar MP3s dentro. Criar pasta THUMBS separada com imagens numeradas. Todas com permissão pública.

CONFIGURAR NO N8N

7

Importar workflow

No n8n → 3 pontinhos → Import from File → selecionar workflow.json

8

Configurar credenciais

Clicar em cada node com triângulo vermelho e conectar as credenciais criadas nas etapas anteriores

9

Ativar Execute Once

No node LISTA MP3s → aba Settings → ativar Execute Once (obrigatório)

10

Publicar e testar

Clicar em Publish → clicar no botão Play para testar → aguardar 10-40 minutos → verificar vídeo no YouTube Studio

TODA SEMANA
