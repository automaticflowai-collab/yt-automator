ANTES DE COMEÇAR

1

Criar conta no n8n

Acesse app.n8n.io → Sign Up → confirme o email

2

Chave API do Claude

console.anthropic.com → API Keys → Create Key → salve sk-ant-... → adicione U$5 em Billing

3

Google Cloud — APIs e OAuth

Criar projeto → ativar YouTube Data API v3 + Gmail API + Drive API → criar chave de API → criar OAuth Client ID com redirect https://app.n8n.io/rest/oauth2-credential/callback

4

Shotstack

shotstack.io → criar conta → copiar chave sandbox

5

Cloudinary

cloudinary.com → criar conta → copiar Cloud Name + API Key + API Secret → criar upload preset sem assinatura

GOOGLE DRIVE

6

Estrutura de pastas

Criar pasta NOME_DO_CANAL → subpastas com data DDMMAAAA → MP3s dentro. Pasta THUMBS separada. Todas com permissão pública.

CONFIGURAR NO N8N

7

Importar workflow

No n8n → 3 pontinhos → Import from File → selecionar workflow.json

8

Configurar credenciais

Clicar em cada node com triângulo vermelho e conectar as credenciais criadas

9

Ativar Execute Once

No node LISTA MP3s → aba Settings → ativar Execute Once (obrigatório)

10

Publicar e testar

Clicar em Publish → Play → aguardar 10-40 minutos → verificar no YouTube Studio

ROTINA DE PUBLICAÇÃO

✓

Antes do dia de publicação

Criar áudios → baixar em MP3 → criar pasta com data no Drive → subir MP3s → subir thumbnail no Cloudinary → atualizar ID da pasta no node LISTA MP3s

🤖

No horário programado

O workflow roda sozinho e posta automaticamente no YouTube com título, descrição e tags gerados pela IA
